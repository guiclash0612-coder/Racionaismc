Racionais MC's — Project scaffold

Instruções rápidas:

1. Coloque a fonte real Wednesday.ttf em public/fonts/Wednesday.ttf (arquivo .ttf)
2. Substitua as imagens em public/assets/ por imagens reais (hero.jpg, album1.jpg ...)
3. No diretório do projeto execute:

   npm install
   npm run dev

Observações:
- Alguns arquivos são placeholders (SVG inside .jpg files). Substitua por imagens reais para ver o visual completo.
- Este scaffold usa React + Vite + Tailwind + Framer Motion + GSAP.
